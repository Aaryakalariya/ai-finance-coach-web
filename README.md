An **AI-powered Personal Finance Management Web Application** developed as a **Semester 5 Mini Project**. This project combines **Full Stack Web Development** and **Artificial Intelligence** to help users manage their finances more effectively.

AI Finance Coach enables users to securely manage their income, expenses, savings, and budgets while receiving intelligent financial guidance through an AI-powered chatbot. The platform is designed to make personal finance simple, organized, and smarter for everyone.

---

# 📌 Project Objectives

- Help users manage their daily finances efficiently.
- Track income and expenses in one place.
- Set monthly spending limits and budgets.
- Provide AI-powered financial guidance.
- Analyze spending habits and generate insights.
- Encourage smarter saving and budgeting habits.

# 📖 How It Works

1. User creates an account.
2. User securely logs into the application.
3. User adds income and expenses.
4. User sets a monthly spending limit.
5. The system automatically tracks spending.
6. Financial reports and analytics are generated.
7. Users can interact with the AI chatbot for personalized financial guidance and recommendations.
